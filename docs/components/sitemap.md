# Sitemap — Визуальная карта сайта

**Файл:** `UX/sitemap.html`  
**Тип:** Страница (Page)  
**Роль:** Интерактивная визуальная архитектура сайта Eplug в виде колонок с эффектами

---

## Описание

Полноэкранная страница с визуальным представлением структуры сайта. Отображает все основные разделы в виде колонок с карточками. Карточки реагируют на hover — есть анимации и выделение. Точка входа для навигации по всем wireframe-страницам.

---

## Структура / Состав

- **5 колонок** — по числу основных разделов (Home, About, Network, Partners, Contact)
- **25 секций** — карточки страниц и подстраниц
- Верхний хедер через `shared-header.js`
- Hover-эффекты на каждой карточке

---

## Навигация

Является главной точкой входа — `UX/index.html` редиректит сюда автоматически.

---

## Зависимости

- `shared-header.js` — общая шапка навигации
- Встроенные стили (все CSS внутри файла, нет внешних `.css`)

---

## Известные ограничения

- Статический HTML — при добавлении нового раздела нужна ручная правка файла
- Большой размер файла (29 KB) из-за инлайн-стилей и разметки
