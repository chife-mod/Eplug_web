# Shared Header — Архитектурный блок

**Файл:** `shared-header.js` (корень проекта)  
**Тип:** Переиспользуемый функциональный модуль (логика навигации)

---

## Описание

`shared-header.js` — единый навигационный блок, который подключается на все страницы проекта через `<script>`. При загрузке он динамически вставляет HTML-разметку шапки в DOM и активирует нужный пункт меню в зависимости от текущего URL.

Это архитектурное решение — **Single Source of Truth** для хедера: один файл управляет навигацией всего сайта.

---

## Принцип работы

1. Страница подключает `<script src="/shared-header.js"></script>`
2. Скрипт создаёт DOM-элемент с разметкой хедера
3. Определяет текущую страницу по `window.location.pathname`
4. Добавляет CSS-класс `.active` к нужному пункту меню
5. Вставляет хедер в начало `<body>`

---

## Зависимости

- Нет внешних зависимостей
- Требует наличия корневого CSS со стилями `.header`, `.nav-link`, `.active`

---

## Известные ограничения

- Изменение структуры навигации требует правки одного файла `shared-header.js`
- При использовании на GitHub Pages нужно учитывать базовый путь (`/Eplug_web/`)
